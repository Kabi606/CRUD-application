CRUD-application-DRF-
CRUD Application (Django REST Framework) – A RESTful API project built with Django and DRF supporting Create, Read, Update, and Delete operations. Includes serializers, viewsets, and router-based endpoints. Ideal for learning API development and practicing CRUD with Django.

Clone the repository
git clone https://github.com/Kabi606/CRUD-application.git

Install dependencies
pip install -r requirements.txt

Apply migrations
python manage.py migrate

Create a superuser
python manage.py createsuperuser

Run the development server
python manage.py runserver

Usage

Open the admin panel at http://127.0.0.1:8000/admin/

Manage employees, assign them to groups, and configure permissions

Extend the system to include employee reports, dashboards, or API endpoints
